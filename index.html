INSTAFIND

Instafind is an object location system which uses radio frequency technology to look for lost items. The product includes tags - which the users can attach to the desired objects, 5 landmarks - which can be placed at any desired location and a central hub - which communicates with the landmarks and the iOS application.

The landmarks have a finder attached to it which transmits a search RF signal to locate the desired object within a 6 feet radius. The tags receive the search RF signal and responds to the finder by signaling, which may be by transmitting a found RF signal if the transmitted and stored tag identification codes match. The finder signals to the central hub and the user informing that the object has been located, and indicates the landmark it was located near

Users use the iOS application built for this product to go over the items list in their catalogue and to locate an object.

Hardware

A core requirement for Systems Design Engineering is knowledge of hardware computing. It is very essential to understand how digital systems work and how they can communicate with software. I build many projects which highlight this field. One of the projects I made is a Home Security System, built using the knowledge of arduino microcontrollers and radio frequency.

The project had a set up of two arduino microcontrollers in a master-slave configuration to enable bidirectional communication for the simulation, and a third arduino acting as a central hub to detect any changes in the sensor platform. The slave arduino would sense the human motion and transmit the data to master arduino, and the master arduino activate the alarm system by sending the central hub a binary message, increasing the frequency with which the LEDs light up.
